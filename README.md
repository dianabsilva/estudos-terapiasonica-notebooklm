# estudos-terapiasonica-notebooklm
Estudo científico sobre Arrastamento de Ondas Cerebrais (BWE) e Terapia Sônica. Analisa o impacto de ruídos coloridos e batimentos binaurais na cognição e saúde usando o NotebookLM. Contém guia de engenharia de prompts (RAG), glossário técnico e matriz comparativa de modulação cortical para otimizar foco e relaxamento.
